# Apollo Subgraph Vapor

This boilerplate is build with:

- Apollo subgraph
- Graphql
- Prisma
- Typegraphql-Prisma

If it helps you, please feel free to give it a start :)

## How to use?

1. Clone repo:
`git clone git@github.com:salman-codewar/apollo-subgraph-vapor.git`

2. Install packages
`yarn install`

3. Prisma migrate
`yarn migrate`

4. Run server
`yarn dev`
